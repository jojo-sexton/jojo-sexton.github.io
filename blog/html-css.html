What does it mean to display inline vs inline blocks?

Inline , Block and Inline-Block in CSS explained in simple terms:

  • {display:inline;} arranges the elements horizontally and doesn't care about top and bottom margins & paddings
  • {display:block;} arranges the elements vertically. The elements do not sit next to each other The
  • {display: inline-block;} arranges the elements horizontally and evenly. This display also allow us to set the width, height and padding of the container of the elements
